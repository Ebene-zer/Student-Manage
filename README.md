# Student Management System (Mini Project)

## Problem Statement:
```
A basic school Records Officer find it challenging in manually taking, keeping, retrieving and updating students details.
```

## Proposed Solution:
```
A simple computer program that allows the Records Officer to Add, update, retrive and delete student details.
```

## Key Features:
- Add new student 
- Update student details
- Search for existing student
- Delete

Add Student:

- ID
- Name
- Age
- Course of Study
- Residential Status
- Hall of Residence if student is boarder

Search Student:
- Search for student using ID or Name

